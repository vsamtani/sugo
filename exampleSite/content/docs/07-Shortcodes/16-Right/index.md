The `right` shortcode right-aligns the wrapped content block. This is similar to the [centered](../07-Centered/) shortcode.

## Shortcode Example

{{< codeblock lang="md" >}}
{{</* right */>}}
This line appears right-aligned.
{{</* /right */>}}
{{< /codeblock >}}

## Actual Use

{{< right >}}
This line appears right-aligned.
{{< /right >}}
