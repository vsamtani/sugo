The `highlight` shortcode renders a centered, large typographic emphasis block. It is intentionally text-only (no card background), so it reads like a pull-quote paragraph.

## Shortcode Example

{{< codeblock lang="md" >}}
{{</* highlight */>}}
This line is highlighted for emphasis.
{{</* /highlight */>}}
{{< /codeblock >}}

## Actual Use

{{< highlight >}}
This line is highlighted for emphasis.
{{< /highlight >}}
