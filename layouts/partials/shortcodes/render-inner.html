{{- $ctx := .ctx -}} {{- $content := .content | default "" -}} {{- $display := .display | default "block" -}} {{- $trim := .trim | default false -}} {{- $rendered := "" -}} {{- if $trim -}} {{- $content = strings.TrimSpace $content -}} {{- end -}} {{- if and $ctx $ctx.Page -}} {{- $rendered = printf "%s" ($ctx.Page.RenderString (dict "display" $display) $content) -}} {{- else -}} {{- $rendered = printf "%s" ($content | markdownify) -}} {{- end -}} {{- $rendered = replaceRE `\[\^([^\]]+)\]` `$1` $rendered -}} {{- $rendered | safeHTML -}}
